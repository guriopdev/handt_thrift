**HandT Thrift** is a premium, hyperlocal peer-to-peer fashion ecosystem built specifically for the **NIT Hamirpur** collegiate community. By restricting access locally, students can securely declutter their wardrobes, organically discover pre-loved clothing, negotiate counter-offers natively in-app, and entirely bypass online transaction algorithms by authorizing final deals locally. 

No shipping logistics. No hidden checkout fees. Pure aesthetic campus connectivity. ✨

---

## 🚀 Features

- 📍 **GPS-Gated Locality:** Access is algorithmically tied strictly to the NIT Hamirpur physical campus 5-kilometer boundaries.
- 💬 **Live Negotiation Engine:** Instead of standard checking out, buyers can seamlessly send, receive, and approve direct counter-price offers through an integrated active real-time chat interface. 
- 🔒 **Verified Student Identity:** Every account requires the explicit generation of a digital HandT Identity Card mapping standard geolocation data and contact pipelines.
- 👗 **Curated Seamless Dashboard:** A strictly chronological and completely unified `Browse -> Chat -> Sell -> Cart -> Wishlist -> Pay` progression meticulously constructed across a fluid singular dashboard.
- 🎨 **Minimalist Aesthetic:** Styled directly with modern frosted glassmorphism, pastel gradients, dynamic micro-animations, and pure CSS marquees.

---

## 🛠️ Technology Stack

This application takes immense architectural advantage of modern web framework infrastructure:

* **Frontend Framework:** `Next.js 14` / `React 18`
* **Styling & Theming:** `Tailwind CSS`, `Lucide-React` (Icons)
* **Authentication & Backend Data:** `Supabase` (Google OAuth)
* **Security Constraints:** Edge rendering and strictly enforced Supabase Row Level Security (RLS) Postgres Database policies.
